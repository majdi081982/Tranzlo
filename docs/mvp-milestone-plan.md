# MVP Milestone Plan

## Milestone 1: Platform foundation

- Monorepo and environment contracts
- Next.js shell and global layout
- Appwrite integration layer
- Docker Compose and Nginx baseline

## Milestone 2: Acquisition and onboarding

- Homepage and pricing
- Signup, login, OAuth redirect wiring
- Role selection and trial activation
- Marketing SEO metadata

## Milestone 3: Marketplace core

- Translator profile management
- Company profile and job posting
- Job search and job detail
- Job applications and applicant review

## Milestone 4: Operations

- Notifications center
- Conversation lists and message views
- Billing and subscription status pages
- Admin moderation, verification, and disputes

## Milestone 5: Content and community

- Blog listing and post detail
- Admin blog CRUD foundation
- Community feed and comments

## Milestone 6: Automation and launch hardening

- n8n workflow deployment
- payment and lifecycle webhooks
- audit logging
- smoke tests, observability, and launch checklist
